Navigation Menu Example

How to integrate this menu:

1. Copy the CSS from the <style> section that contains the navigation styles
2. Copy the HTML for the navigation menu (the <nav class="main-nav"> element)
3. Paste these into your ai_features.html file
4. Update the links to match your actual routes
